PROJECT TITLE: Dodgeit
PURPOSE OF PROJECT: To create a game with a Hero dodging the Bananas flying at it.
VERSION or DATE: 2024.04.26
HOW TO START THIS PROJECT: First, clone all the files to a local repository. Second, open the file named "project.greenfoot". The game windows will show up to you after these steps.
AUTHORS: Felix Huang
USER INSTRUCTIONS: The game will start when you click the "Run" button. You can click on the background to move the "Hero" upward and downward to dodge the Bananas moving toward it. If the Hero is hit by a Banana, the game will stop and a Sad Face and the text "Game Over!" will pop up to you.
